To start developing on PartyMaps release:

1. Install android sdk
	http://developer.android.com/sdk/index.html#win-bundle
2. Configure the Eclipse ADT-Plugin
	http://developer.android.com/sdk/installing/installing-adt.html

3. Open Eclipse->File->New->Project->Other->Android Project From Existing Code
4. Browse on your directory to find "Project for Eclipse" and select Finish
5. Your'e good to go
	5.1 Code on /exemple/platforms/android/assets/www
	5.2 Execute with Eclipse execution module for Android
